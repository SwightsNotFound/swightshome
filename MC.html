How to join The Vanilla Minecraft server

(Do note that you need to have been added to the whitelist to join.)

Recommended method

You will need a custom Minecraft launcher, I would recommend either Atlauncher or Prism Launcher.

Once installed, download a client modpack here, then import it into your launcher, once done simply boot up the new instance and press play in the menu.

These mods are all completely client side and help boost your in-game performance, it also has the one required mod Modflared.

Manual method

This server uses Cloudflare Tunnels in order to forward the server hosted on my machine to yours, however due to limitations with Cloudflare we have to use a mod called Modflared.

I would recommend a custom Minecraft launcher to make this easier, either Atlauncher or Prism Launcher.
