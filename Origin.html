How to join The Origins Minecraft server

(Do note that you need to have been added to the whitelist to join.)

You will need a custom Minecraft launcher, I would recommend either Atlauncher or Prism Launcher.

Once installed, download the server's modpack here, then import it into your launcher, once done simply boot up the new instance and press play in the menu.
